# Python Voice Assistant (Jarvis)

A Python-based voice assistant capable of recognizing voice commands and responding using text-to-speech.

## Features
- Voice command recognition
- Text-to-speech responses
- Opens websites and performs basic tasks
- Executes predefined commands

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## How to Run
1. Install Python
2. Install required libraries:
   pip install -r requirements.txt
3. Run the program:
   python jarvis.py

## Note
This project was built as part of learning Python automation and voice-based applications.
